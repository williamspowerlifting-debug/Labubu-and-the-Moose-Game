# Labubu & Moose Adventure

Retro-inspired browser platform game.

## Features

- Play as Labubu or Moose
- Pixel-art style graphics
- Mobile touch controls
- Enemies and coins
- Retro sound effects
- Side-scrolling gameplay
- GitHub Pages compatible
- Fully serverless

## Deployment

Upload all files to a GitHub repository root.

Enable:

Settings -> Pages -> Deploy from branch -> main -> /(root)

Your game will be live at:

https://YOUR_USERNAME.github.io/REPO_NAME/
